# Iremintas pasisveikinimas (1 oji užduotis)

![version][version-badge]

## Programos tikslas

-Atspausdinti ireminta varda.
-Leisti pasirinkti remelio dydi.
-Skirtingai kreiptis i moteri ar vyra.

## Idiegimas (Unix kompiuteryje)

- `git clone https://github.com/vikuliukas/OOP_1_uzd.git`
- `cd OOP_1_uzd`
- `make`
- `./pasisveikinimas`
